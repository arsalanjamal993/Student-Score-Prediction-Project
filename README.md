# Student Score Prediction Project

A machine learning project built with **Python** and **scikit-learn** that predicts student scores based on study hours. This project demonstrates the use of **linear regression** to model real-world data, perform predictions, and evaluate model performance.

## 📝 Project Overview

The project uses historical data of students’ study hours and their corresponding scores to build a predictive model. The trained model can estimate the expected score for a given number of study hours. This showcases practical knowledge of:

- Data preprocessing
- Linear regression
- Model training and testing
- Predictions and evaluation

## 📊 Features

- **Input Data:** Number of study hours per student.
- **Output Prediction:** Predicted score based on study hours.
- **Model Evaluation:** Calculates accuracy metrics like R² score and Mean Absolute Error.
- **Visualization:** Optional plotting of regression line and data points using Matplotlib.

## 💻 Technologies Used

- **Python 3.x**
- **pandas** – For data manipulation
- **NumPy** – For numerical computations
- **scikit-learn** – For machine learning model creation and evaluation
- **Matplotlib / Seaborn** – For data visualization
